Установка и настройка WP

Установка WP на localhost

  1. Скачать, установить и запустить XAMPP
  2. Создать базу данных в http://localhost/phpmyadmin/
  3. Скачать WordPress и распаковать архив в C:\xampp\htdocs\имя-сайта
  4. В файле wp-config-sample.php изменить 3 строки:
    define( 'DB_NAME', 'имя-базы-данных' );
    define( 'DB_USER', 'root' );
    define( 'DB_PASSWORD', '' );
  5. Переименовать wp-config-sample.php в wp-config.php
  6. Открыть в браузере http://localhost/имя-сайта/ и заполнить форму

Перенос WP с localhost на хостинг

  1. Залить все файлы в корень хостинга
  2. Импортировать базу данных из XAMPP на хостинг
  3. Выполнить SQL запрос по смене адреса http://localhost/имя-сайта (без слеша в конце). Сервис генерации запроса.
  4. В файле wp-config.php изменить имя базы, логин и пароль
